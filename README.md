## 準備
### 環境の準備
```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

mecabのインストールにつまづくかもしれません。
それぞれの環境に合わせたインストールを行なってください。
https://taku910.github.io/mecab/#install
